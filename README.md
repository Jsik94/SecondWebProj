Kosmo 9th web project

## 목적 ✨
- 웹 프로젝트를 완성
- 부트스트랩 Template 적용해볼 것
- CRUD 구현
- CP로 DB 연결할 것
- 모델지향적으로 소통할것
- 스크립트를 통해 
- 페이지네이션,좋아요, 댓글 만들어보기
